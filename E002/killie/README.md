# Killie

Killie is a little package that shows a warning for old IE
browser users

Usage

    :html
    <html>
      <head>
        <script src="http://cdn.lovely.io/core.js"></script>
        <script src="http://cdn.lovely.io/killie.js"></script>
      </head>
    </html>
